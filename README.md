# Computer Vision & Image Processing Foundations

## Project Overview
This project explores the technical fundamentals of digital image processing using Python. In the context of computer vision and machine learning, images are treated as complex, multi-dimensional numerical matrices. This project showcases my ability to manipulate raw pixel arrays, apply structural transformations, and preprocess visual data—a critical pipeline step for training downstream deep learning models like Convolutional Neural Networks (CNNs).

## Tech Stack & Key Libraries
* **Language:** Python
* **Core Frameworks:** `NumPy` (for matrix and array slicing), `Matplotlib` (for side-by-side visual reporting), `OpenCV` / `Pillow` (for image loading and processing).
* **Environment:** Jupyter Notebook / VS Code

## Key Technical Transformations Implemented
* **Multi-Dimensional Array Slicing:** Manipulated raw image matrices to extract regions of interest, alter pixel structures, and reshape array dimensions.
* **Color Space Conversion:** Implemented transformations (such as converting RGB/BGR data into Grayscale) to optimize data structures for computational efficiency.
* **Feature Enhancement & Filtering:** Executed pixel-level mathematical operations to adjust brightness, modify contrast ratios, and apply basic filters.
* **Visual Validation Plots:** Built side-by-side plotting structures to map and document the direct relationship between programmatic matrix modifications and the resulting visual output.
